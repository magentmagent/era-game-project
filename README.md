# eraKaihen

**Emuera1821+v8 기반** 성인 Era 텍스트 게임

> 모험자를 자신의 길드에 가입시킨 후, 최면을 걸어 전투를 성행위/성봉사로 상식개변시키고 던전에 보내 몬스터들에게 성처리를 시키며, 그 보수로 금품을 받는 거대 길드 운영 게임

## 🎯 게임 개요

- 길드 주인: 당신
- 목표: 모험자를 최면으로 만들어 던전에 보내 몬스터의 성처리를 이용해 수익 창출
- 핵심: 다양한 상식개변 패턴 + 직업별 음란 스킬 변환
